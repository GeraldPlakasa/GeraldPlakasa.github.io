The Problem

Optimal Cleaning Path

Bayangkan robot vacuum di ruangan dengan multiple spots of dirt. Bagaimana ia memutuskan kemana harus bergerak selanjutnya?

Classic AI Problem: Vacuum World adalah benchmark problem di AI—simple tapi illustrative untuk konsep agent dan environment.
The Algorithm

Greedy Approach

Agent menggunakan strategi greedy: di setiap langkah, pilih action yang memberikan immediate benefit terbesar.

  • Sense: Detect semua dirt dalam environment
  • Calculate: Hitung jarak ke setiap dirt
  • Choose: Pilih dirt terdekat (greedy choice)
  • Act: Move ke target dan clean
  • Repeat: Sampai semua bersih
Key Learnings

Greedy ≠ Always Optimal

Project ini mengajarkan trade-off fundamental dalam AI:

  • Fast decisions: Greedy sangat cepat—O(n) per decision
  • Local optimum: Bisa stuck di solusi yang "bagus tapi bukan terbaik"
  • Real-world relevance: Robot vacuum sungguhan pakai similar heuristics
  • Foundation: Stepping stone ke algorithms lebih kompleks (A*, genetic, dll)
