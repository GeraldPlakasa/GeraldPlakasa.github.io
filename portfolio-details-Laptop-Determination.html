The Project

AHP-Based Decision Making

Mahasiswa baru sering bingung memilih laptop. Sistem ini menggunakan metode Analytic Hierarchy Process (AHP) untuk memberikan rekomendasi berdasarkan kriteria yang diprioritaskan user.

  • Multi-criteria: Storage, harga, performa, dan kriteria lainnya
  • Weighted Scoring: User bisa set prioritas masing-masing kriteria
  • Web Interface: Django-based UI yang user-friendly
