The Project

Web Scraping + Text Segmentation

Proyek ini mengeksplorasi teknik ekstraksi informasi dari web dan membandingkan efektivitas berbagai metode segmentasi teks.

  • Punkt: NLTK sentence tokenizer berbasis unsupervised learning
  • Split: Simple string splitting berdasarkan delimiter
  • Tokenize: Word-level tokenization untuk analisis granular
