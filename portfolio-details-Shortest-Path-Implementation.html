The Project

BFS vs DFS Comparison

Proyek ini mengimplementasikan dan membandingkan dua algoritma graph traversal klasik untuk mencari jalur terpendek.

  • BFS: Breadth-First Search—garanteed shortest path untuk unweighted graphs
  • DFS: Depth-First Search—memory efficient tapi tidak optimal
  • Game Application: Enemy AI pathfinding ke posisi player
