The Vision

Dunia yang Hidup Sendiri

Ashen World bukan game biasa—ini adalah simulasi sosial dimana setiap villager adalah AI dengan personality traits unik yang mempengaruhi keputusan mereka.

Core Philosophy: Pemain tidak mengontrol langsung, tapi menyaksikan drama yang unfold secara organik. Villager bisa jadi pahlawan, pengkhianat, atau korban—tergantung trait dan circumstance mereka.
Game Systems

15+ Interlocking Systems

Setiap sistem saling mempengaruhi, menciptakan emergent gameplay yang tidak pernah sama dua kali.

  • Elections & Leadership: Villager voting untuk King setiap 5 tahun, dengan term limit dan corruption mechanics
  • Family & Inheritance: Marriage, children, family tree visualization, dan inheritance system
  • Economy & Buildings: Tax policy, treasury management, construct/upgrade/repair buildings
  • Social Dynamics: Relationships, friendships, rivalries, assassination plots
  • Weather System: Sunny/rain affecting villager actions and productivity
  • Combat & Enemies: External threats, combat resolution, hero villagers
Architecture

Clean Code, Full Coverage

Project ini dibangun dengan arsitektur yang proper: separation of concerns, repository pattern, dan comprehensive test suite.

  • Repository Pattern: Data persistence layer terpisah dari business logic
  • Service Layer: 11 services untuk different game systems
  • TypedDict Models: Type safety untuk semua data structures
  • 181 Tests: Unit + integration tests dengan pytest
  • Real-time API: JSON API untuk auto-refresh dashboard
Features

Full-Featured Web Dashboard

Semua diakses via browser—dark theme UI dengan real-time updates.

  • Kingdom Dashboard: Sortable/filterable villager table dengan pinned character card
  • Admin Control: Reset world, tick days, start auto-simulation
  • Player Character: Create your own character dan watch them live their life
  • Family Tree: vis-network visualization untuk family relationships
  • Leaderboard: Yearly champions dan all-time records
