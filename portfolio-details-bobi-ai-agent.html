Capabilities

What Bobi Can Do

Dengan akses ke VPS dan multiple tools, Bobi bisa melakukan berbagai task secara autonomous.

  • Reminders & Scheduling: Set reminders, manage calendar, dan notify via WhatsApp
  • File Management: Create, edit, organize files dan folders di server
  • Content Posting: Post ke social media, manage content calendar
  • Chat & Communication: Reply messages, handle conversations dengan context
  • Build & Code: Write code, create scripts, deploy applications
  • Research: Deep research workflows dengan web browsing capability
Architecture

Dual-Brain System

Bobi menggunakan kombinasi dua AI model untuk balance antara speed dan capability.

  • Codex (ChatGPT): Fast responses untuk daily tasks dan conversations
  • Claude (Anthropic): Deep reasoning untuk complex tasks dan coding
  • OpenClaw Platform: Framework yang menghubungkan AI dengan tools dan services
  • Memory System: Persistent memory untuk continuity across sessions
